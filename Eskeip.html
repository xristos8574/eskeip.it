What is Eskeip.it!?

Eskeip.it! is an advanced, cross-platform browser-based reconnaissance and kiosk testing utility. It is designed to probe the capabilities, permissions, and vulnerabilities of a web browser environment across desktop, iOS, and Android platforms.

The tool's primary purpose is to assist security researchers and system administrators in identifying potential "escapes" from restricted browser environments (such as kiosk modes or sandboxed tablets) and to perform comprehensive reconnaissance on the hosting hardware and OS via standard Web APIs.

How it can be used:

  • Kiosk Testing: Attempting to break out of restricted interfaces by triggering OS-level dialogs, print previews, or deep links to system settings.
  • Hardware Recon: Accessing and testing connected peripherals like cameras, microphones, USB devices, and sensors.
  • Network Discovery: Identifying local IP addresses, scanning for local network devices, and testing connectivity to common router gateways.
  • Fingerprinting: Generating unique device fingerprints to understand how much identifying information is exposed to a website.
  • Platform Verification: Checking which advanced Web APIs (like NFC or Web Serial) are enabled on a target device.
